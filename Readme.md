Light Date Status
Plugin ultra-léger pour Obsidian pour afficher dates de création et modifiaction dans la barre de statut Obsidian.

Affichage heure 24H Francais.

Informations :
- Affichage Discret : Les dates apparaissent en bas à droite de l'interface. Voir image du dépot pour screenshot
- Respect de vos données : Le plugin ne modifie pas le texte de vos notes (pas de YAML, pas de texte caché).
- Format Local : Affichage au format français (24h) : `JJ/MM/AAAA HH:mm`.
- Zéro Latence : Optimisé pour ne pas ralentir votre coffre (Vault).

Installation
1. Téléchargez "main.js" et "manifest.json" depuis https://github.com/Live2Di3/Obsidian_Light_Date_Status/releases/latest
2. Créez un dossier "Light_Date_Status" dans ".obsidian/plugins/
3. Copiez les deux fichiers à l'intérieur.
4. Activez le plugin dans les réglages d'Obsidian.
